# [Sample Size Estimator](https://bowei-zhang.shinyapps.io/sample_size_estimator/)

## A tool for calculating sufficient Sample Size for A/B Testing

### Input:  
*Historical Average Metrics*  
*Lift Threshold*  
*Confidence Level*  
*Number of Groups*    


### Output:  
*Sample Size*  
*Test Duration(optional)*
